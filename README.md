# proxmox-tools

Collection of scripts and configs for **Proxmox homelab**.

## Scripts

- `jellyfin_gpu_efficiency` – Measure frames per watt for iGPU transcoding
- `jellyfin_ram_transcode` – Set Jellyfin transcode folder in RAM
- `cpu_tuning` – CPU undervolt and pstate tuning scripts
- `powertop_autotune` – Apply powertop auto-tune for power efficiency
